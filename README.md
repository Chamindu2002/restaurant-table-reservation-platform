# Restaurant Table Reservation Platform

## 📌 Project Overview
This is a web-based Restaurant Table Reservation Platform developed using Java JSP/Servlet architecture. The system allows users to create, view, update, and delete restaurant table reservations.

## 🚀 Features
- Create table reservations
- View all reservations
- Update reservation details
- Delete reservations
- Input validations
- Reservation cost calculation
- File handling persistence

## 🛠 Technologies Used
- Java
- JSP
- Servlets
- Apache Tomcat
- Maven
- HTML/CSS
- Bootstrap
- JavaScript
- File Handling

## 📂 OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## 📁 Project Structure
- Model Layer
- Service Layer
- Utility Layer
- Servlet Controller
- JSP Frontend

## 💾 Data Storage
Reservation data is stored using text file handling.
